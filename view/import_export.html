Import and Export

You can import and export settings. Only Json format is supported.
hover over each button to see a description.
The settings are saved in the order of row1, row2...

設定のインポート・エクスポートが可能です。各ボタンにホバーすることで説明を見れます。
形式はJsonのみ対応しています。row1, row2...の順番に設定が保存されます。

Format

                
                  {
                    "row1": {
                      "url_row_1": "https://www.example.com/",
                      "css_selector_row_1": ".top-navigation",
                      "color_row_1": "#746cea",
                      "service_row_1": "none",
                      "id_row_1": ""
                    },
                    "row2": {
                      "url_row_2": "https://*.console.aws.amazon.com/",
                      "css_selector_row_2": ".globalNav-223",
                      "color_row_2": "#ff0000",
                      "service_row_2": "aws",
                      "id_row_2": "012345678912"
                    }
                  }
