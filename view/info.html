Description of Color Changer

Color Changerの説明

URL

In the URLfield, specify the URL of the web page for which you want to change the color. You can also use regular expressions.

URL には、色を変更したいウェブページのURLを指定します。正規表現が使えます。

Example1 (例): 　https://en.wikipedia.org/wiki/Test

Example2 (例): 　https://en.wikipedia.org/wiki/*　←'*' for all wiki pages

CSS Selector

The CSS Selector field specifies a CSS selector. This allows you to change the background of the page as well as the color of certain parts of the page. What is CSS Selector??

CSSセレクタ には、CSSセレクタを指定します。これによりページの背景の他、特定の部分だけ色を変えることができます。 CSSセレクタについて

Example1 (例): 　body　← changes background color

Example2 (例): 　.mw-header　← changes header of wiki pages

Color

In the Color field, you can specify a color. Click to select a color.

Color フィールドでは色を指定できます。クリックによりカラーを選択できます。

Service and ID

The Service and ID field are configured for AWS and Azure users; AWS users can specify an account ID and Azure users can specify a subscription ID.

ServiceおよびID フィールドは、AWSおよびAzure使用者のために設定されています。AWSであればアカウントID、AzureであればサブスクリプションIDを指定できます。

Actions

You can add a new rule by clicking the plus icon (+). You can delete an existing rule with the trash can icon (🗑).

Please don't forget to hit the save button after you added your settings!

プラスアイコン (+) をクリックして新しいルールを追加できます。ゴミ箱アイコン (🗑) で既存のルールを削除できます。

ルールを追加した後は保存ボタンを押して適用します。

Import and Export

You can import and export in Json format. See the Import/Export section for details.

Json形式でインポートやエクスポートができます。詳しくはImport Exportページを参照ください。
